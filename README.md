# Lab Experiment Shell Scripts

A collection of shell scripts created for lab experiments, testing, automation, and command-line practice.

These scripts are mainly used for:
- Linux lab tasks
- Bash scripting practice
- Automation experiments
- System command testing
- Educational purposes

---

## Features

- Simple Bash scripts
- Lightweight and fast
- Easy to understand
- Beginner-friendly
- Useful for lab assignments and experiments

---

## Requirements

- Linux / macOS / WSL
- Bash shell

Check Bash version:

```bash
bash --version
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
```

Navigate into the project directory:

```bash
cd your-repo
```

Give execution permission:

```bash
chmod +x *.sh
```

---

## Usage

Run any script using:

```bash
./script.sh
```

Or:

```bash
bash script.sh
```
